# AI-Powered Log Analyzer (Mini SIEM)

## 📌 Overview
This project simulates a SIEM (Security Information and Event Management) tool used in SOC environments. It analyzes logs, detects threats, and provides a dashboard for security monitoring.

## 🚀 Features
- Threat Detection (Brute Force, SQL Injection)
- Event Correlation (Repeated attack detection)
- AI-Based Anomaly Detection (Isolation Forest)
- Threat Scoring System
- Streamlit Dashboard for visualization
- Top Attacker IP Tracking
- Log Search Functionality
- Export Alerts as CSV

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn

## ▶️ How to Run
```bash
pip install pandas scikit-learn streamlit
python -m streamlit run app.py
